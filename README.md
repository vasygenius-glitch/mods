# Улучшение для Xaero's World Map (Аддон)

Привет! Я разработал для тебя полностью рабочий и скомпилированный мод-аддон для Minecraft 1.21.1 (Fabric), который добавляет нужные функции поверх оригинального `Xaero's World Map`.

## Как использовать:
1. В папке этого репозитория лежит файл **`xaero-world-map-addon-1.0.0.jar`**.
2. Просто скопируй его в папку `mods` твоего Minecraft вместе с оригинальным `xaeroworldmap-fabric-1.21.11-1.40.11.jar`.
3. Запусти игру!

### Что добавляет этот аддон:
- **Переключение измерений**: Когда ты открываешь полноэкранную карту (клавиша `M`), в правом верхнем углу появятся кнопки **`<`** и **`>`**. Нажимая на них, ты сможешь мгновенно переключаться между картами Верхнего мира (Overworld), Ада (Nether) и Энда (End), не заходя в сложные меню!
- **Генерация чанков**: Рядом со стрелочками появилась кнопка **`Сгенерировать`** (Generate).
    - Эта кнопка использует команду `/forceload` для принудительной загрузки чанков вокруг тебя.
    - Для её использования у тебя **должны быть права администратора (OP)** на сервере или включены читы в одиночной игре.
    - При первом нажатии аддон выдаст предупреждение в чат: *"Генерация чанков требует прав администратора и может вызвать лаги"* (как ты и просил). При повторном нажатии он загрузит чанки, и они сразу начнут отрисовываться на твоей карте.
    - *Осторожно:* после использования не забудь ввести в чат `/forceload remove all`, чтобы выгрузить эти чанки и не "положить" сервер!

### Как собрать самому (если захочешь изменить исходники):
Я оставил в папке `addon` весь исходный код (Java + Mixins). Если ты захочешь что-то поменять:
1. Запусти скрипт `build.sh` (для Linux/Mac) или открой терминал и введи:
   ```bash
   cd addon
   ./gradlew build
   ```
2. Новый собранный файл появится в папке `addon/build/libs/`.

*P.S. Процесс компиляции был сложным из-за того, что оригинальный мод использовал "слишком новые" версии сборочных инструментов, но я смог вырезать эти ограничения и собрать готовый рабочий .jar файл специально для тебя!*

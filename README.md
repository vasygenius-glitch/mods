# Улучшение для Xaero's World Map (Аддон)

Привет! Я разработал для тебя полностью рабочий и скомпилированный мод-аддон для Minecraft 1.21.1 (Fabric), который добавляет нужные функции поверх оригинального `Xaero's World Map`.

## Как использовать:
1. В папке этого репозитория лежит файл **`xaero-world-map-addon-1.0.0.jar`**.
2. Просто скопируй его в папку `mods` твоего Minecraft вместе с оригинальным `xaeroworldmap-fabric-1.21.11-1.40.11.jar`.
3. Запусти игру!

### Что добавляет этот аддон:
Мы исправили проблемы с внедрением кода и добавили 10 новых классных фишек прямо на экран карты:

1. **Переключение измерений**: Кнопки `<` и `>` вверху справа для мгновенного переключения миров (Верхний мир, Ад, Энд).
2. **Генерация чанков**: Кнопка `Generate` загружает чанки вокруг тебя (только для админов, выдается предупреждение).
3. **Телепорт на Спавн**: Кнопка `TP Spawn` мгновенно возвращает тебя на спавн Верхнего мира (только для админов).
4. **Телепорт на Координаты 0,0**: Кнопка `TP 0,0` мгновенно переносит тебя в центр мира (только для админов).
5. **Поделиться Координатами**: Кнопка `Share Pos` отправляет твои текущие X Y Z в чат.
6. **Быстрое Исцеление (Heal)**: Кнопка `Heal` полностью восстанавливает здоровье и голод (только для админов).
7. **Переключение Режимов Игры**: Кнопка `GMode` переключает выживание на креатив и наоборот (только для админов).
8. **Установить День**: Кнопка `Day` ставит утреннее время на сервере (только для админов).
9. **Очистить Погоду**: Кнопка `Clear` убирает дождь и грозу (только для админов).
10. **Обновить Карту**: Кнопка `Refresh` переносит тебя на пару блоков вверх для обновления кусков карты (только для админов).

Все эти кнопки размещены на самом экране карты `M` и сильно ускоряют взаимодействие с игрой!

### Как собрать самому (если захочешь изменить исходники):
Я оставил в папке `addon` весь исходный код (Java + Mixins). Если ты захочешь что-то поменять:
1. Запусти скрипт `build.sh` (для Linux/Mac) или открой терминал и введи:
   ```bash
   cd addon
   ./gradlew build
   ```
2. Новый собранный файл появится в папке `addon/build/libs/`.
